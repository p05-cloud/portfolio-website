# Portfolio Website

A responsive, professionally designed portfolio showcasing expertise in **Cloud Architecture**, **DevOps Engineering**, and **Enterprise Infrastructure**.

Built to present technical capabilities, hands-on experience, and continuous professional development in a clear, recruiter-friendly format.

---

## Live Website

**[View Portfolio →](https://portfolio-website-ks.vercel.app/)**

---

## Site Structure

| Page | File | Description |
|------|------|-------------|
| **Home** | `index.html` | Landing page with profile snapshot, tech stack, and navigation |
| **About Me** | `about_me.html` | Background, interests, and professional focus |
| **Skills** | `skills.html` | Technical competencies across cloud, DevOps, and security |
| **Learning** | `learning.html` | Continuous development initiatives and study tracks |
| **Projects** | `project.html` | Cloud implementations and automation workflows |
| **Experience** | `experience.html` | Professional experience with defined outcomes |
| **Education** | `education.html` | Academic background and formal training |
| **Certifications** | `certificate.html` | Industry certifications from AWS, IBM, and HashiCorp |
| **Achievements** | `achievements.html` | Milestones, recognition, and professional accomplishments |

---

## Tech Stack

**Frontend**
- HTML5, CSS3, JavaScript
- Custom responsive layouts
- CSS animations and transitions

**Design**
- Dark theme with cyan/magenta accents
- Mobile-first responsive design
- Consistent component styling

**Deployment**
- Static site hosting
- Compatible with GitHub Pages, Vercel, Netlify

---

## Key Features

- **Responsive Design** — Optimized for desktop, tablet, and mobile
- **Profile Snapshot** — Quick overview of role, focus areas, and metrics
- **Terminal Animation** — Dynamic CLI-style display on homepage
- **Tech Stack Tags** — Visual representation of technical expertise
- **Timeline Layouts** — Structured presentation for projects and experience
- **Fast Load Times** — No heavy frameworks, pure HTML/CSS/JS

---

## Local Development

```bash
# Clone the repository
git clone https://github.com/p05-cloud/portfolio-website.git

# Navigate to project
cd portfolio-website

# Open in browser
open index.html
```

No build process required — fully static site.

---

## Deployment

This site is deployment-ready for any static hosting platform:

| Platform | Command / Method |
|----------|------------------|
| **Vercel** | Connect repo → Auto-deploy |
| **GitHub Pages** | Settings → Pages → Select branch |
| **Netlify** | Drag & drop or connect repo |
